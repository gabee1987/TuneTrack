Privacy Policy Repository

About this helper site

This small site hosts the privacy policy and helpful links for building and testing TuneTrack. It is styled to match the app theme (colors and feel) so testers and reviewers see a consistent brand.

Privacy-first

TuneTrack does not request microphone access in production. Camera is only used for gameplay and QR scanning.

Testing

Use the Play Console internal testing track or build a preview via EAS to verify permissions and installation.

No monetization

This project is intended for private, non-commercial use.
